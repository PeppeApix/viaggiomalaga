  • La spiaggia più lunga di Malaga è Playa de la Malagueta, ideale per rilassarsi.

How This Website Was Built with HTML

What is HTML?

HTML (HyperText Markup Language) is the standard language for creating web pages. It uses tags to structure content like headings, paragraphs, images, and links. For example: 

<h1>This is a Heading</h1>
<p>This is a paragraph.</p>
<img src="photo.jpg" alt="A beautiful view">

Key HTML Features I Used

  • Sections & Navigation: I organized content with <section> tags and linked them to the menu using <a href="#id">.
  • Multimedia: I embedded a <video> of my trip and a Google Maps <iframe> for the itinerary.
  • Tables: The statistics section uses <table> to display data clearly.
  • CSS Styling: I added colors, fonts, and responsive design with <style> in the <head>.

Why HTML Matters

Learning HTML helped me create this website from scratch. It's the foundation of all web development—combined with CSS and JavaScript, it brings designs to life!

Architettura Client-Server

Come funziona questo sito web

Questo sito utilizza il modello client-server, alla base di Internet:

  • Client (il tuo browser): Richiede e visualizza la pagina usando HTML/CSS/JavaScript.
  • Server: Memorizza i file del sito e li invia quando richiesto.
  • Protocollo HTTP/HTTPS: Il "linguaggio" usato per comunicare.

Passaggi tecnici

Quando hai aperto questa pagina:

  1. Il tuo browser (client) ha inviato una richiesta al server.
  2. Il server ha elaborato la richiesta e inviato i file HTML/CSS.
  3. Il browser ha interpretato il codice mostrandoti questa pagina.

Database e Memoria dei Dati

Cosa sono i Database?

Un database è un sistema organizzato per archiviare, gestire e recuperare informazioni. Nel contesto web, i database memorizzano tutti i dati dinamici di un sito.

Esempi di dati che potrei memorizzare:

  • Recensioni dei visitatori su Malaga
  • Prenotazioni per tour guidati
  • Statistiche aggiornate sui turisti
  • Preferenze degli utenti

Vantaggi dei database:

  • Dati organizzati e facilmente accessibili
  • Possibilità di aggiornamenti in tempo reale
  • Sicurezza e backup delle informazioni
  • Ricerca e filtraggio efficiente

Tipi di Database

Tipo Descrizione Esempi
Relazionali Dati organizzati in tabelle collegate MySQL, PostgreSQL
NoSQL Dati non strutturati o semi-strutturati MongoDB, Firebase
Memoria locale Dati salvati direttamente nel browser LocalStorage, IndexedDB

Esempio Pratico per Questo Sito

Anche se questo sito è statico, potrei implementare un database per:

Struttura Tabella "Recensioni"

CREATE TABLE recensioni (
    id INT AUTO_INCREMENT PRIMARY KEY,
    nome VARCHAR(50),
    email VARCHAR(100),
    valutazione INT,
    commento TEXT,
    data_inserimento DATETIME
);

Query di Esempio

-- Inserire una nuova recensione
INSERT INTO recensioni 
(nome, email, valutazione, commento, data_inserimento)
VALUES 
('Mario Rossi', 'mario@example.com', 5, 
 'Bellissima città! Tornerò sicuramente', NOW());

-- Recuperare le recensioni migliori
SELECT * FROM recensioni 
WHERE valutazione >= 4 
ORDER BY data_inserimento DESC 
LIMIT 5;
